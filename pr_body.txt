Summary

This branch adds a set of targeted backend unit tests, test helpers, and a CI workflow to measure/enforce coverage.

What changed
- New test helpers: pkg/app/testutils/test_helpers.go
- New backend tests:
  - tests/pvc_files/pvc_files_test.go
  - tests/k8s_graph/builder_test.go
