# KubeDevBench
KubeDevBench is a Kubernetes client application designed to provide developers and DevOps engineers with a comprehensive set of features for managing Kubernetes clusters.

Vibecoded.

## Stack
- Frontend: React
- Backend: Go
- Application Framework: Wails (https://wails.io/)

## Building

```bash
wails build
```

## Running

```bash
wails dev
```
